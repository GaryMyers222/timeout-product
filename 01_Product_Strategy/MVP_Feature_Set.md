\# TimeOut MVP Feature Set



The MVP must enable the full circle loop:



Invite Friends → Send Sit Request → AutoPing → Sit Happens → Points Transfer



Modules:



1\. Phone number identity

2\. Circle creation \& invites

3\. Sit request entry

4\. AutoPing engine

5\. SMS \& push notifications

6\. Sit completion

7\. Points ledger



Secondary MVP features:



\- Emergency Daycare Pickup

\- Playdate broadcast

\- TimeOut gathering RSVP
