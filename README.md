# TimeOut

TimeOut is a mobile app that helps trusted friends trade babysitting through a private, invitation-only circle.

The system coordinates sit requests among nearby parents so families can enjoy occasional TimeOut while maintaining reciprocity and trust.

TimeOut is **not a babysitter marketplace** and does not connect strangers.  
It is designed for groups of friends who already trust one another.

---

# Core Idea

Parents often struggle to find a reliable sitter on short notice.

TimeOut removes the friction by coordinating requests among a small group of trusted friends.

Core loop:

Invite Friends → Send Sit Request → AutoPing finds sitter → Sit happens → Points transfer → Circle grows stronger

---

# Key Features

### Invitation-Only Circles
Parents create a private TimeOut circle with friends they trust.

### AutoPing
When a sit request is created, the system quietly contacts sitter-friends until the first **YES** is received.

### Points Ledger
Points help maintain reciprocity between members without turning the system into a formal market.

### Emergency Daycare Pickup
Urgent requests broadcast to the circle with an incentive for fast responses.

### Playdates
Members can broadcast playdate invitations and gather multiple RSVPs.

### TimeOut Gatherings
Periodic gatherings help circles maintain trust, socialize, and recruit thoughtfully.

### Circle Vital Signs
The system can monitor activity and gently nudge circles to maintain healthy participation.

---

# Product Philosophy

TimeOut is built around a simple principle:

**Friends helping friends.**

The software should remain simple so the circle itself becomes the real support system.

Design priorities:

- low friction for tired parents
- trusted relationships
- minimal typing and screens
- fast coordination through messaging
- long-term community continuity
